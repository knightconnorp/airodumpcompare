# airodumpcompare.py
Process and compare default airodump-ng CSV mac address prefixes against 'mac-vendors-export.csv' (from maclookup.app) showing associated organization/device type<br>

## Steps
Step 1 - Run airodump-ng (or collect other CSV with mac addresses as first column<br>
Step 2 - Run airodumpcompare.py<br>

## Options
-h OR --help    Help menu<br>
-u              Show UNKNOWN mac addresses in results<br>
-o              Skip trimming CSV file, allows for other than airodump-ng CSV files<br>

## Dependencies
- None
